# Indian-Banks-Dashboard-2021-25
Developed an interactive Power BI dashboard to analyze the financial performance of major Indian banks from 2021–2025. The dashboard provides insights into market share, revenue growth, net profit trends, and equity analysis using dynamic visualizations and KPI reporting.

Implemented year-wise growth analysis, profit comparison charts, and market share visualization to evaluate banking performance across multiple financial institutions. Applied DAX calculations, custom formatting, and interactive slicers for efficient financial reporting and business intelligence analysis.

The Indian Banks Financial Analysis Dashboard is an interactive Power BI project designed to analyze and compare the financial performance of major Indian banks between 2021 and 2025. The dashboard helps visualize market share, revenue trends, net profit growth, and equity performance using business intelligence and data analytics techniques.

This project transforms banking financial data into meaningful visual insights to support comparative financial analysis and strategic decision-making.
